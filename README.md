# jlu_helper

这是一个力图整合您在吉林大学校园中生活需要的绝大多数信息服务的项目。

## 缘由

得益于我吉无数先辈们的无私奉献，在github上存在众多的针对学校各个系统的脚本，但由于其使用存在一定的技术门槛且大多由不同的同学开发比较分散，故一直想做这样一个项目将各位大佬的成果结合起来。
